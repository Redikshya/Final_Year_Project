About Lockify

Lockify is an intelligent access management system designed to provide secure and seamless entry for authorized personnel. By combining advanced recognition technologies with a user-friendly interface, it ensures efficient monitoring and control of premises. The system enhances security, reduces unauthorized access, and provides detailed logs for administrative oversight.

How to Navigate

To explore the project, use the main landing page:

  • Door Access: Click to enter the access control panel and verify entry permissions.
  • Admin: Click to log in as an administrator to manage users and view access logs.
  • Use the Back button to return to the home page at any time.
